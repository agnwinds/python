Diag.angle
================

This option defines the angle from which all the photons are emitted in searchlight mode. 0° is at the pole. The default value is 0°.

Type
  Double

Units
  Degrees

Values
  Greater than 0

File
  `diag.c <https://github.com/agnwinds/python/blob/master/source/diag.c>`_
