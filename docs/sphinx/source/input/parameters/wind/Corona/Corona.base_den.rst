Corona.base_den
===============
The coronal model is defined in terms of a base density and a scale height. ``base_den`` ask for the base density of the corona.

Type
  Double

Unit
  :math:`number/cm^3`

Values
  Greater than 0

File
  `corona.c <https://github.com/agnwinds/python/blob/master/source/corona.c>`_
