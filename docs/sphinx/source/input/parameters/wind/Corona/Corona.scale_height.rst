Corona.scale_height
===================
The coronal model is defined in terms of a base density
and a scale height. ``scale_height`` asks for the scale height of the corona. 

Type
  Double

Unit
  cm

Values
  Greater than 0

File
  `corona.c <https://github.com/agnwinds/python/blob/master/source/corona.c>`_
